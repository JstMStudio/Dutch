Over

JstMStudio is een kleine studio opgericht door Jst Tan. JstMStudio is op veel dingen gericht, van muziek, het schrijven van verhalen, website-ontwikkeling en nog veel meer. JstMStudio is een kleine afgelegen indiestudio die zich op veel dingen richt om een betere toekomst op te bouwen.

Bijdragen aan JstMStudio

Er zijn veel dingen die je zou kunnen helpen en bijdragen aan JstMStudio-projecten en de twam helpen. Elke hulp en bijdrage aan onze projecten wordt door ons op prijs gesteld, of het nu gaat om een kleine typefout of een fout van het team.



Bugs rapporteren

Een manier om zonder technische kennis bij te dragen aan het project, is door bugs aan het team te melden. Als je een bug op onze website of software hebt gevonden, maak dan een Github-probleem aan of stuur ons een e-mail en meld het. Het zou ons enorm helpen door onze software te verbeteren, zodat iedereen deze soepeler kan gebruiken.

Als je zou kunnen, zou je ons ook kunnen helpen met het oplossen van gerapporteerde bugs wanneer we de bugs nog niet hebben opgelost. Het zou het team zeker enorm helpen om tijd en moeite te besparen bij het vinden van bugs. Nadat u klaar bent met programmeren, kunt u een pull-verzoek indienen voor een van onze vrijwilligers om het goed te keuren en te implementeren.



Inhoud toevoegen aan JstMStudio-software

U kunt bijdragen aan het project door inhoud toe te voegen aan onze software. U kunt sites of projecten indienen bij onze zoekmachine, video's, muziek en boeken toevoegen aan onze inhoudsbibliotheek zodat iedereen ervan kan genieten, enzovoort.

Doneren

JstMStudio wordt gefinancierd door mensen zoals jij. We verkopen geen gebruikersgegevens, spammen uw scherm niet met advertenties of vergrendelen dingen in de betaalmuur. Verlenging

JstMStudio-profiel is de allereerste sociale site die daadwerkelijk ondersteuning biedt voor extensies op gebruikersprofielen en berichten. Een extensie is een software-add-on die op een programma wordt geïnstalleerd en de mogelijkheden ervan verbetert. Gebruikers konden kiezen uit verschillende extensies om in hun profiel te gebruiken en in de JstMStudio Profile-extensiebibliotheek te posten. De JstMStudio-profielextensie is gebouwd door het officiële personeelsteam en de community.

Gebruikers kunnen alle plug-ins hier in hun code installeren door deze toe te voegen voordat de berichtelementen eindigen of op hun profiel. Installeer de plug-in door de onderstaande code te kopiëren en te plakken.



Klap

Clap is een van de officiële extensies van het JstMStudio-profiel. Het kan worden gebruikt in berichten van JstMStudio Profile-gebruikers. Gebruikers konden applaudisseren wanneer deze plug-in was geïnstalleerd.



Reageer

Commentaar is een van de officiële extensies van het JstMStudio-profiel. Gebruikers kunnen ze aan hun bericht toevoegen zodat de kijker er commentaar op kan geven. Gebruikers kunnen het ook gebruiken om feedback op hun bericht te krijgen.



Vind ik leuk

Like is een van de officiële extensies van het JstMStudio-profiel. Gebruikers konden ze aan hun post toevoegen zodat kijkers en schrijvers hun reactie op de post konden laten zien. Gebruikers kunnen ook reageren om hun steun te betuigen.



Delen

Share is een van de officiële extensies van het JstMStudio-profiel. Gebruikers kunnen ze aan hun post toevoegen zodat kijkers hun post op sociale media kunnen delen. Gebruikers kunnen ook reageren om hun steun te betuigen door het bericht te delen.



Omhoog

Updown is een van de officiële uitbreidingen van het JstMStudio-profiel. Gebruikers konden ze aan hun berichten toevoegen zodat kijkers en schrijvers hun reacties op het bericht konden laten zien. Gebruikers konden ook reageren om hun steun te betuigen. Updown kan ook worden gebruikt voor peilingen.

API

JstMStudio leverde veel services en functies binnen onze platformsoftware, JstMStudio App Platform. JstMStudio API is een Application Programming Interface voor JstMStudio App Platform die ondersteuning biedt aan alle ontwikkelaars om diensten zoals in JstMStudio App Platform in hun website te implementeren zonder de functies zelf volledig gratis en zonder kosten te ontwikkelen. Het is gebouwd voor iedereen om te gebruiken en niet om geld te verdienen van gebruikers zoals ontwikkelaar en website-eigenaar.

JstMStudio API is een open source API ( Application Programming Interface ). Iedereen kon alle JstMStudio API-broncodes bekijken, lezen en bestuderen op onze officiële Github-repository. JstMStudio API is gebouwd met behulp van HTML, CSS en vanille JavaScript, in tegenstelling tot andere traditionele API's, en gelicentieerd onder AGPL 3.0.
